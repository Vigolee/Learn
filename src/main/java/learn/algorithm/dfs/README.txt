DFS从某个状态开始，不断地转移状态直到无法转移，然后退回到前一步的状态，继续转移到其他状态，
不断重复，直到找到最终解（例如：数独）。根据DFS特点，采用递归函数实现比较简单。

1、LakeCount
    问题:有一个大小为N*M的园子，雨后积起了水。八连通的积水被认为是连接在一起的。请求出院子里共有多少水洼？（八连通指的是下图相对W的.部分）
    输入:10 12
               W........WW.
               .WWW.....WWW
               ....WW...WW.
               .........WW.
               .........W..
               ..W......W..
               .W.W.....WW.
               W.W.W.....W.
               .W.W......W.
               ..W.......W.
    输出：3
    说明：‘W’表示积水，‘.’表示没有积水
